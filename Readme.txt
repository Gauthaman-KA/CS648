***The directory structure has to be replicated without changes for the 'Pipeline.ipynb' file to execute correctly.***

The code for the dissertation 'Enhancing Energy Market Operations and Planning by Leveraging Ensembled Models' is available in a single Jupyter Notebook file named 'Pipeline.ipynb' in the directory 'CodeFiles'.

Additional .ipynb files are available for each individual model studied containing their hyperparameter tuning procedures.
